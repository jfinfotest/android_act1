# Taller de Fundamentos de Kotlin

Este proyecto contiene la resolución de 30 ejercicios prácticos para consolidar los conocimientos en el lenguaje Kotlin.

## Descripción del Proyecto
El proyecto está organizado en 6 bloques temáticos que cubren:
1. **Variables y Tipos de Datos:** Manejo de `val`, `var`, tipos explícitos e inferencia.
2. **Null Safety:** Uso de operadores safe call `?.`, elvis `?:`, `let` y aserción `!!`.
3. **Condicionales:** Estructuras `if` y `when`, rangos y verificación de tipos.
4. **Ciclos:** Bucles `for` (con `step` y `downTo`), `while` y `do-while`.
5. **Funciones:** Definición de funciones, parámetros por defecto, argumentos nombrados y funciones de expresión única.
6. **Colecciones y Extensiones:** Listas mutables, filtrado, búsqueda y creación de funciones de extensión.

## Cómo Ejecutar
Para ejecutar el taller, inicia la clase `Main.kt`. Esto ejecutará secuencialmente todos los bloques de ejercicios y mostrará los resultados en la consola.
